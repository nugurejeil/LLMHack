# Product Requirements Document
## TOEFL 80 - 바쁜 직장인을 위한 3개월 토플 학습 앱

| 항목 | 내용 |
|------|------|
| 문서 버전 | 2.0 |
| 작성일 | 2024년 12월 |
| 작성자 | 한재현 (CPO, Weniv) |
| 상태 | Draft |

---

## 1. Executive Summary

TOEFL 80은 풀타임 직장인이 3개월 안에 토플 80점대를 달성할 수 있도록 설계된 모바일 학습 앱입니다. 평일 9-6 근무, 육아 등 시간적 제약이 있는 사용자가 출퇴근 시간, 점심시간, 아이들 취침 후 등 자투리 시간을 활용하여 효율적으로 학습할 수 있습니다.

---

## 2. Problem Statement

### 2.1 현재 시장의 문제점

- 대부분의 토플 학습 앱은 풀타임 학생을 대상으로 설계됨
- 학습 세션이 30분~1시간 이상으로 직장인이 활용하기 어려움
- 진도 관리가 비현실적 (하루 2-3시간 학습 가정)
- 가족이 있는 사용자의 라이프스타일 고려 부족

### 2.2 목표 사용자 페르소나

| 항목 | 내용 |
|------|------|
| 이름 | 김현우 (가상 페르소나) |
| 나이 | 38세 |
| 직업 | IT 기업 개발자 / 10년 경력 |
| 가족 | 배우자 + 자녀 2명 (7세, 영아) |
| 근무시간 | 평일 09:00 - 18:00 |
| 영어 수준 | 과거 토익 800 (10년 이상 공백) |
| 학습 목표 | 해외 대학원 진학 (TOEFL 79점 이상 필요) |
| Pain Points | 시간 부족, 배우자 부담 최소화, 스피킹/라이팅 두려움 |

## 3. Product Goals

### 3.1 핵심 목표

1. **3개월 내 TOEFL 80점대 달성** - 기존 영어 기초가 있는 사용자 기준
2. **하루 1시간 이내 학습으로 목표 도달** - 자투리 시간 활용 극대화
3. **가족 친화적 학습 설계** - 배우자/가족에게 부담 주지 않는 학습 패턴

### 3.2 성공 지표 (KPIs)

| 지표 | 목표 | 측정 방법 |
|------|------|----------|
| 일일 학습 완료율 | 80% 이상 | 일일 목표 달성 비율 |
| 주간 리텐션 | 70% 이상 | 7일 연속 접속 비율 |
| 목표 점수 달성률 | 60% 이상 | 3개월 후 실제 토플 점수 |
| 평균 세션 시간 | 10-15분 | 세션 분석 |

## 4. Design Concept

### 4.1 디자인 철학

**"따뜻하고 귀여운 학습 친구"**

딱딱하고 스트레스를 주는 기존 학습 앱과 차별화하여, 사용자가 매일 열고 싶은 아늑하고 친근한 앱을 지향합니다. 수채화 느낌의 일러스트와 따뜻한 색감으로 학습 부담을 줄이고, 귀여운 마스코트 캐릭터가 학습 여정을 함께합니다.

### 4.2 컬러 팔레트

| 용도 | 컬러명 | HEX | 설명 |
|------|--------|-----|------|
| Primary | Warm Cream | `#FDF6E3` | 메인 배경색, 부드러운 크림색 |
| Secondary | Soft Peach | `#FFECD2` | 카드 배경, 따뜻한 피치색 |
| Accent 1 | Strawberry Pink | `#E8A0A0` | 강조, 버튼, 딸기 핑크 |
| Accent 2 | Honey Brown | `#D4A574` | 포인트, 따뜻한 꿀갈색 |
| Accent 3 | Cocoa Brown | `#8B6F5C` | 텍스트, 진한 코코아색 |
| Success | Mint Green | `#A8D5BA` | 정답, 완료 상태 |
| Warning | Butter Yellow | `#F4D03F` | 알림, 주의 |
| Text Primary | Warm Black | `#3D3D3D` | 본문 텍스트 |
| Text Secondary | Soft Gray | `#7A7A7A` | 보조 텍스트 |

### 4.3 마스코트 캐릭터

학습 섹션별로 귀여운 동물 캐릭터가 사용자를 안내합니다.

| 캐릭터 | 이름 | 담당 섹션 | 특징 |
|--------|------|----------|------|
| 🐹 햄스터 | 모찌 (Mochi) | 단어 학습 | 머리에 딸기를 올린 통통한 햄스터, 단어를 볼에 가득 저장 |
| 🐱 고양이 | 토스티 (Toasty) | 리딩 | 식빵 모양 고양이, 버터를 올린 따뜻한 빵 컨셉 |
| 🐻‍❄️ 북극곰 | 코코 (Coco) | 리스닝 | 니트 스웨터를 입고 코코아를 마시는 북극곰 |
| 🦊 여우 | 러스티 (Rusty) | 스피킹 | 마이크를 들고 있는 수줍은 여우 |
| 🐰 토끼 | 페니 (Penny) | 라이팅 | 작은 안경을 쓰고 펜을 든 토끼 |

### 4.4 일러스트레이션 스타일

- **스타일:** 따뜻한 수채화 느낌의 손그림 스타일
- **선:** 부드러운 연필 텍스처의 아웃라인
- **색감:** 채도를 낮춘 파스텔톤, 자연스러운 그라데이션
- **질감:** 종이 텍스처 배경, 수채화 번짐 효과

### 4.5 타이포그래피

| 용도 | 폰트 | 스타일 |
|------|------|--------|
| 한글 제목 | Pretendard | Bold, 둥근 느낌 |
| 한글 본문 | Pretendard | Regular |
| 영어 제목 | Quicksand | Bold, 둥근 산세리프 |
| 영어 본문 | Nunito | Regular |
| 숫자/강조 | Baloo 2 | 귀여운 둥근 폰트 |

### 4.6 UI 컴포넌트 스타일

**카드**
- 둥근 모서리 (border-radius: 16px~24px)
- 부드러운 그림자 (box-shadow: soft, warm tint)
- 크림/피치 배경색

**버튼**
- 둥근 pill 형태
- Primary: Strawberry Pink + 흰색 텍스트
- Secondary: Honey Brown + 흰색 텍스트
- 호버 시 부드러운 스케일 애니메이션

**아이콘**
- 둥근 선 스타일 (rounded stroke)
- 2px 두께
- 캐릭터와 어울리는 손그림 느낌

**프로그레스 바**
- 둥근 모서리
- 그라데이션 채우기 (Peach → Pink)
- 완료 시 반짝이는 애니메이션

**입력 필드**
- 둥근 모서리
- 크림색 배경
- 포커스 시 Peach 테두리

### 4.7 애니메이션 & 인터랙션

- **캐릭터 반응:** 정답 시 캐릭터가 기뻐하는 모션, 오답 시 격려 모션
- **전환:** 부드러운 fade, slide 트랜지션 (300ms)
- **마이크로 인터랙션:** 버튼 탭 시 살짝 튀어오르는 효과
- **로딩:** 캐릭터가 걸어다니는 로딩 애니메이션
- **축하:** 목표 달성 시 색종이/별 파티클 효과

### 4.8 화면별 디자인 가이드

**홈 화면**
- 오늘의 마스코트가 인사하는 헤더
- 따뜻한 색감의 학습 카드 그리드
- 스트릭 불꽃 + 캐릭터 축하 메시지

**학습 화면**
- 해당 섹션 캐릭터가 상단에 표시
- 문제 카드는 부드러운 크림색 배경
- 정답 시 캐릭터 + 칭찬 메시지 팝업

**결과 화면**
- 점수에 따른 캐릭터 표정 변화
- 따뜻한 격려 메시지
- 부드러운 색상의 통계 차트

## 5. Core Features

### 5.1 Micro-Learning 시스템

모든 학습 콘텐츠를 5-15분 단위의 마이크로 세션으로 분할하여 자투리 시간 활용을 극대화합니다.

- **Quick Session (5분):** 단어 10개, 문법 포인트 1개
- **Standard Session (10분):** 리딩 1지문, 리스닝 1강의
- **Deep Session (15분):** 스피킹/라이팅 연습

### 5.2 Time-Aware 스케줄링

사용자의 일정에 맞춰 자동으로 최적의 학습 시간과 콘텐츠를 추천합니다.

| 시간대 | 추천 콘텐츠 | 이유 |
|--------|------------|------|
| 새벽 (5:30-7:00) | 라이팅, 리딩 | 집중력 최고, 조용한 환경 |
| 출퇴근 (7:00-9:00) | 리스닝, 단어 | 이어폰 착용, 이동 중 |
| 점심 (12:00-13:00) | 단어 복습, 문법 | 짧은 시간, 가벼운 학습 |
| 밤 (21:00-22:30) | 스피킹 | 아이들 취침 후 |

### 5.3 AI 스피킹 코치

음성 인식 기반의 스피킹 연습 기능으로, 실시간 피드백을 제공합니다.

- **발음 분석:** 실시간 발음 정확도 피드백
- **템플릿 제공:** 80점 목표에 최적화된 답변 구조

### 5.4 적응형 학습 엔진

사용자의 실력과 약점을 분석하여 개인화된 학습 경로를 제공합니다.

- **진단 테스트:** 최초 가입 시 현재 수준 측정
- **약점 분석:** 섹션별 세부 역량 분석
- **동적 난이도:** 정답률에 따른 난이도 자동 조절
- **주간 리포트:** 진도 및 예상 점수 분석

## 6. 12주 커리큘럼 구조

### 6.1 Phase 1: 기초 회복 (Week 1-4)

- 토플 형식 익히기 + 기본 어휘/문법 복습
- 일일 목표: 리스닝 30분, 단어 20개, 리딩 1지문
- 주간 목표: 모의 리딩 1회, 리스닝 1회

### 6.2 Phase 2: 약점 집중 (Week 5-8)

- 개인별 약점 섹션 집중 공략
- 스피킹/라이팅 본격 시작
- 주간 목표: 풀 섹션 모의고사 1회

### 6.3 Phase 3: 실전 대비 (Week 9-12)

- 풀 모의고사 + 약점 보완
- 시간 관리 전략 훈련
- 주간 목표: 풀 모의고사 1-2회
